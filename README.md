# OpenVPN Two-Factor Authentication (2FA) Extensions

These plugins and scripts can be used to add support for additional two-factor authentication (2FA) methods to an OpenVPN server.

## Documentation & Tutorials

Documentation and tutorials for setting up two-factor authentication using these plugins and scripts [can be found here](https://www.sparklabs.com/support/kb/category/two-factor-authentication-setup-guides/), while guides for setting up an OpenVPN server on a large number of platforms [can be found here](https://www.sparklabs.com/support/kb/category/vpn-server-setup-guides/).

## FIDO/U2F Two-Factor Authentication Plugin

This plugin enables FIDO/U2F authentication tokens and PAM (Pluggable Authentication Modules) to be used to authenticate users on an OpenVPN server. A user without an associated token will be automatically prompted to register their token when they first connect to the OpenVPN server. They can then authenticate with this token when connecting to the OpenVPN server. An associated device for a user can be reset using the included script.

For a detailed setup guide please see [U2F Two-Factor Authentication with OpenVPN and Viscosity](https://www.sparklabs.com/support/kb/article/yubikey-u2f-two-factor-authentication-with-openvpn-and-viscosity/).

## TOTP Two-Factor Authentication Script

This script enables Time-based One-Time Password (TOTP) codes and PAM (Pluggable Authentication Modules) to be used to authenticate users on an OpenVPN server. TOTP codes can be generated using almost all popular password managers and two-factor code generators.

For a detailed setup guide please see [TOTP Two-Factor Authentication with OpenVPN and Viscosity](https://www.sparklabs.com/support/kb/article/totp-two-factor-authentication-with-openvpn-and-viscosity/).

## YubiKey OTP Two-Factor Authentication Script

This script enables the One-Time Password (OTP) feature on YubiKeys and PAM (Pluggable Authentication Modules) to be used to authenticate users on an OpenVPN server. A YubiKey can be registered to a user, and OTP codes can be generated by touching the button on the YubiKey token when prompted.

For a detailed setup guide please see [YubiKey OTP Two-Factor Authentication with OpenVPN and Viscosity](https://www.sparklabs.com/support/kb/article/yubikey-otp-two-factor-authentication-with-openvpn-and-viscosity/).

## Duo Two-Factor Authentication Script

This script enables Cisco Duo Two-Factor Authentication and PAM (Pluggable Authentication Modules) to be used to authenticate users on an OpenVPN server. Push authentication to registered devices, passcodes, and phone callbacks are all supported methods.

For a detailed setup guide please see [Duo Security Two-Factor Authentication with OpenVPN and Viscosity](https://sparklabs.com/support/kb/article/duo-security-two-factor-authentication-with-openvpn-and-viscosity/).
